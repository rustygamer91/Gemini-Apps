# Winter Games '26 Dashboard

A professional, interactive analytics dashboard for predicting the 2026 Winter Olympics medal tally. Built as a high-performance micro-tool with no external dependencies.

**[Live Preview](index.html)**

## Features

-   **Multi-Model Simulation**: Toggle between "Gracenote Consensus" (Analyst), "Beijing '22 Repeater" (Historical), and "Host & Momentum" (Speculative) models.
-   **Interactive Visualization**: Segmented "Titans" vs. "Field" view with dynamic bar charts.
-   **Smart Sorting**: Instantly toggle between "By Total" and "By Gold" with a premium segmented control.
-   **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.
-   **Instant Sharing**: Copy results to clipboard with one click.

## Deployment Guide

### Run Locally
Simply open `index.html` in any modern web browser. No compilation or server required.

### Deploy to Vercel
1.  Push this folder to a GitHub repository.
2.  Import the repository in Vercel.
3.  Vercel will automatically detect the static project.
4.  Click **Deploy**.

## Tech Stack
-   **Core**: HTML5, Vanilla JavaScript (ES6+)
-   **Styles**: CSS3 (Glassmorphism, Flexbox, Grid, Variables)
-   **Performance**: Zero external dependencies, <30KB size.
